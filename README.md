# VIP-API-java-client

## Description
It is a java code generated by swagger codegen from the CARMIN specification (version: 0.2.1)

## How to use
To use this API, you have to get your own APIkey by connecting to [VIP portal] (https://vip.creatis.insa-lyon.fr/)
and creating your account.

Each module can be used indenpendently, the processing modul contains the basic funtionality and the
data modul contains all request about file exchange.
